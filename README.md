# RocketTurtle

RocketTurtle bridges the gap between AI-generated software and human understanding.

> AI is the rocket. Humans are the turtle.

RocketTurtle helps developers, AI-assisted builders, and non-traditional coders understand the repositories they build and evolve.

---

## What is RocketTurtle?

RocketTurtle transforms complex GitHub repositories into clear mental models.

Paste a repository URL and RocketTurtle helps you understand:

- how the project works
- which files matter most
- what changed
- what is safe to edit
- where to start learning
- how the architecture is evolving over time

---

## MVP

Paste a public GitHub repository URL and generate a beginner-friendly Repo Understanding Report.

---

## First Features

- Fetch public GitHub repository structure
- Detect frameworks and tech stack
- Identify important files and folders
- Generate repository summaries
- Explain how the application works
- Classify safe vs risky files
- Suggest what to learn or edit first
- Answer repository questions in plain English

---

## Future Vision

RocketTurtle will evolve into a persistent understanding layer for AI-generated software.

Future capabilities may include:

- Repository evolution tracking
- AI-generated change summaries
- Repo memory and context persistence
- Architecture maps and dependency visualization
- AI agent guidance files (`skills.md`)
- Beginner and non-technical explanation modes
- Repository onboarding and learning paths

---

## Philosophy

AI can generate software at rocket speed, but humans still understand gradually.

RocketTurtle helps humans keep up with the code they ship.
