# RocketTurtle

RocketTurtle is a beginner-friendly repository understanding platform.

## MVP Scope

- Paste a public GitHub repository URL
- Generate a calm, beginner-friendly Repo Understanding Report
- Detect stack clues and important files
- Highlight safe vs risky/core files
- Suggest a simple learning order

## Tech

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```
