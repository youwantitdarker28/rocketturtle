# RocketTurtle

RocketTurtle bridges the gap between AI-generated software and human understanding.

> AI is the rocket. Humans are the turtle.

## MVP Scope

RocketTurtle is a beginner-friendly repository understanding platform built with:

- Next.js (App Router)
- TypeScript
- Tailwind CSS

Users can paste a **public GitHub repository URL** and receive a clean Repo Understanding Report with:

- Repository summary
- Detected tech stack
- Important files and folders
- Safe files to edit
- Risky/core files
- Suggested learning order
- Beginner architecture explanation

## Local Development

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Notes

- This MVP intentionally uses placeholder/mock explanations.
- No authentication, database, PDF export, private repo support, or AI chat is included yet.
